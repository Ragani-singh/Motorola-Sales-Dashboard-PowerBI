# Motorola-Sales-Dashboard | Power BI Project

## Project Overview
This project is an interactive Power BI dashboard created using Motorola mobile sales data. The dashboard provides insights into sales performance, customer behavior, product performance, and payment trends.

## Objectives
The main objective of this project is to analyze mobile sales data and identify key business insights such as:

- Total sales performance
- Sales trends by month and day
- Top performing brands and models
- Sales distribution across cities
- Customer ratings analysis
- Payment method analysis

## Tools and Technologies Used
- Power BI
- Power Query
- Microsoft Excel

## Data Cleaning and Transformation (Power Query)
The following data cleaning steps were performed using Power Query:

- Removed null and blank values
- Changed data types for accuracy
- Renamed columns for better readability
- Removed duplicate records
- Transformed raw data into structured format
